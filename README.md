# Dalgopchang 달곱창

Authentic Korean Gopchang Experience - Premium Korean BBQ.

This project is a modern, mobile-first restaurant website for **Dalgopchang**, featuring an image-based menu, premium typography, and a sophisticated dark aesthetic.

## Features

- **Image-Based Menu**: Fully visual menu organized by categories (Combo, BBQ, Sides, Drinks).
- **Premium Design**: Warm cream and dark red color palette with a clear typographic hierarchy.
- **Responsive Layout**: Optimized for all screen sizes, from mobile to desktop.
- **Interactive Elements**: Smooth animations using `motion/react` and a zoomed modal/lightbox for menu items.
- **Sections**:
  - Hero with high-impact visuals.
  - About Us story.
  - Visual Image Gallery.
  - Location & Contact with interactive map placeholder.
  - Special Deals & Promotions.
  - Blog/Journal for brand stories.
  - Digital Gift Card section.

## Tech Stack

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Motion/React** (Animations)
- **Shadcn UI** (Components)

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
